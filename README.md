# HeapVisualizer2.0

HeapVisualizer2.0 is a tool to visualize and manage heap data structures.

## Features

- Heap visualization
- Tree structure display
- Unit testing
- Docker support
- Continuous Integration with GitHub Actions

## Installation

Follow the instructions in [docs/INSTALL.md](docs/INSTALL.md).

## Usage

```bash
python heap_visualizer.py
```
